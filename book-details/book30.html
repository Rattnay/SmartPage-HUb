A web application consists of three primary components:

  • Definition: The user-facing side of the application. It includes everything the user sees and interacts with, such as buttons, forms, animations, and layouts.

    • Languages: JavaScript (Node.js), Python (Django, Flask), PHP, Ruby on Rails
    • Frameworks: Express.js (Node.js), Spring Boot (Java), Laravel (PHP)

    • Cloud Databases: AWS RDS, Google Firestore, Supabase
